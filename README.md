# Yachi是怎麼樣的平台呢？
  *內容僅供學術研究使用
  
 1. Yachi是一個整合資訊技術的平台，這裡面包含了電子優惠券、數據分析，讓夜市商家導入這些技術，並能推動精準行銷。
 2. 可以在Yachi找到夜市商家的詳細資訊、這裡面包涵了菜單、價格以及使用者的評價，也是一個主打使用夜市優惠券的平台，減少傳統紙本消費券的浪費及不方便。
 3. 更可以追蹤自己喜愛的店家以及收藏喜愛優惠券，更快找到自己喜愛的店家和優惠券。
